Sistem Informasi Penjualan Bibit Kelapa Sawit di UD. Putri Hijau

Sistem Informasi Penjualan Bibit Kelapa Sawit di UD. Putri Hijau merupakan sistem berbasis web yang dibuat untuk membantu proses penjualan bibit kelapa sawit yang sebelumnya dilakukan secara manual. Sistem ini memungkinkan pengelolaan data penjualan, stok bibit, pengguna, serta pembuatan laporan dilakukan secara terstruktur, cepat, dan akurat. Sistem dapat diakses oleh admin dan pembeli melalui internet sehingga memudahkan proses transaksi dan penyampaian informasi.

Fitur Sistem
Fitur-fitur yang tersedia dalam sistem ini meliputi:
1. Login dan Registrasi Pengguna
Memungkinkan admin dan pembeli untuk masuk ke sistem sesuai dengan hak akses masing-masing.
2. Dashboard Admin
Menampilkan ringkasan informasi seperti total transaksi dan pendapatan, serta menu pengelolaan sistem.
3. Manajemen Data Bibit
Admin dapat menambah, mengubah, dan menghapus data bibit kelapa sawit yang meliputi jenis bibit, harga, stok, dan deskripsi.
4. Manajemen Pengguna
Admin dapat mengelola data pengguna yang terdaftar dalam sistem.
5. Pemesanan Bibit Secara Online
Pembeli dapat melihat katalog bibit dan melakukan pemesanan melalui sistem.
6. Manajemen Pesanan
Admin dapat memproses pesanan, mengubah status pesanan, dan mengonfirmasi pengiriman.
7. Laporan Penjualan
Sistem menyediakan laporan penjualan harian, dan bulanan yang dapat dicetak.

Tujuan Sistem
Tujuan dibuatnya Sistem Informasi Penjualan Bibit Kelapa Sawit di UD. Putri Hijau adalah:
1. Meningkatkan efisiensi dan akurasi dalam proses pencatatan transaksi penjualan bibit kelapa sawit.
2. Mempermudah pengelolaan stok bibit dan data penjualan secara terpusat.
3. Mempercepat proses pelayanan kepada pelanggan melalui pemesanan dan transaksi online.
4. Mengurangi kesalahan pencatatan yang sering terjadi pada sistem manual.
5. Memperluas jangkauan pemasaran UD. Putri Hijau melalui platform berbasis web yang dapat diakses kapan saja dan di mana saja.
